Przygotowanie do WebService


  • Rozgrzewka - dopisać obsługę GET do klienta
  • Napisać serwer HTTP dla kolejki
    • POST /enqueu/:email
    • GET /dequeue
  • Napisać klienta HTTP - dodawanie do kolejki
  • Napisać klienta HTTP - usuwanie z kolejki

Extra

  • Ustawianie url klienta z lini poleceń np. argument --url (hint: flag package)
  • Wczytywanie konfiguracji serwera z pliku json/yaml
  • Whitelist/Blacklist adresów IP po stronie serwera
  • Whitelist/Blacklist adresów email po stronie serwera
  • Hot reload serwera (zero downtime)
